WORKFLOW 1: Customer Registration and Login

Step 1: Customer opens the Workify website
Step 2: Customer clicks on Register button
Step 3: Customer enters name, email, password, phone number
Step 4: System validates input data
Step 5: System creates customer account
Step 6: Customer logs in using email and password
Step 7: System verifies credentials
Step 8: Customer is redirected to dashboard

Features involved:
• User Registration
• User Login
• Authentication
• Customer Dashboard

--------------------------------------------------

WORKFLOW 2: Worker Profile Creation and Service Listing

Step 1: Worker registers on the website
Step 2: Worker logs into account
Step 3: Worker opens profile section
Step 4: Worker enters skills (plumbing, electrical, etc.)
Step 5: Worker enters pricing and location
Step 6: Worker submits profile
Step 7: System saves worker profile
Step 8: Worker profile becomes visible to customers

Features involved:
• Worker Registration
• Profile Creation
• Service Listing
• Profile Management

--------------------------------------------------

WORKFLOW 3: Customer Booking and Review System

Step 1: Customer logs in
Step 2: Customer searches for worker
Step 3: System displays worker profiles
Step 4: Customer selects worker
Step 5: Customer selects date and time
Step 6: Customer submits booking request
Step 7: Worker accepts or rejects booking
Step 8: Service is completed
Step 9: Customer gives rating and review
Step 10: System saves review

Features involved:
• Worker Search
• Booking System
• Booking Status Management
• Review and Rating System
